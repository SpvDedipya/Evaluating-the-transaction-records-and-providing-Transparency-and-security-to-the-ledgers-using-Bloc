Cryptogram



The term Data holds a greater value in the present world which is running after technology. As the saying goes if you are having abundant data, you are powerful enough to rule the world. In this particular circumstance its crystal clear that data can be a boon if proper security is provided or it will be a bane and destroys everything. The government/private organizations in India lack transparency, security and supervision which has marked a negative impact on the willingness of the sponsors or organizations. It has been a simple task to misuse the funds and showing the wrong records to the higher authorities. To avoid these issues Blockchain technology can be proposed as a perfect solution. This particular web application will be validating the transaction, only when the validation is successful from both sides then only the data is sent into the chain and the person can be added into the chain. In this case fake records or unauthorized transactions will not be allowed into the chain. So, this website focusses completely on providing security to the data that you want to keep safe. This provides security and transparency to the data that you want to store adding to that only authorized data is allowed to enter the Blockchain.
